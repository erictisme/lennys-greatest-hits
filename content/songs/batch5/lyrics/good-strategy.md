**Craft Rules:**
- [x] Show don't tell — magnifying glass image, David's stone, the climb. No definitions.
- [x] One killer line — "Every yes you give away / is a no you owe yourself"
- [x] Chorus repeats identically (final chorus changes only "Burn it down" for payoff)
- [x] End-stopped lines — every line is a complete phrase
- [x] No stat-stuffing — "seventeen" is scene-setting, not data
- [x] No insider jargon — David/Goliath is universal
- [x] No thesis statements — no "strategy means" or "strategy isn't"
- [x] Sing test — open vowels on "burn," "thread," "blow," "glow," "ahead"
- [x] Under 3000 chars — ~1400 chars

**What makes this different from Playing to Win (batch 5 song 2):**
- Playing to Win = contemplative folk, reflective, Roger Martin's where-to-play framework
- Good Strategy = neo-soul groove, confident, Rumelt's focus-and-action energy
- Different genre, different tempo, different emotional register
- PtW asks "where do we play?" — GS says "stop asking, start doing"
