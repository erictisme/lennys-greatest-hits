## Chord Progressions

**Key: Dm | BPM: 106**

**Verse: Dm9 - G13 - Cmaj9 - Fmaj7**
Jazzy, unresolved — captures the "floating without purpose" feeling. The extended chords (9ths, 13ths) add sophistication and restless tension, like you know something's off but can't name it. The descent from Dm to Fmaj7 mirrors cascading through tasks without direction.

**Chorus: Bb - C - Dm - F**
Opens up with the IV-V-i movement. More direct and anthemic than the verse — the moment of clarity when you start tracing the line back. The F major at the end lifts before cycling back to Dm, keeping momentum.

**Bridge: Gm7 - A7 - Dm - Bb**
The A7 dominant creates real tension pulling toward Dm — mirrors the "aha" of TED's two-word mission. Resolves to Bb for warmth rather than triumph, so the killer couplet ("biggest missions / smallest task") lands with weight, not bombast.

## Self-Evaluation

### Lyric Craft
- [x] Show don't define — "shipped the button, closed the ticket out" shows meaningless work; Ocean's Eleven shows the framework in action; no thesis statements
- [x] 6-12 syllables per line — all lines 7-10 syllables; anchor phrases ("Trace it back") are intentional repetition
- [x] One killer line — "The biggest missions fit inside your chest / The smallest task should pass the same damn test"
- [x] Chorus varies on repetition — three variations: work context → heist context → personal resolution; last chorus flips from question to answer
- [x] No stat-stuffing — zero numbers crammed into overstuffed lines
- [x] End-stopped lines — every line is a complete phrase, no thoughts dangling across breaks
- [x] Open vowels on key words — "back," "task," "heart," "apart," "chest," "test" all end on singable sounds

### Suno Technical
- [x] Under 3000 characters total (~1100 chars)
- [x] Section tags in [brackets]
- [x] 1 verse before first chorus
- [x] Outro is 2 lines
- [x] Chorus shorter than verse (6 lines each, but chorus has 6-syllable anchor lines making it feel lighter)
- [x] Syllable counts consistent within sections (verse: 9-10, chorus: 7-8 + anchors)
- [x] Style prompt includes vocal character + mood + instruments

### Batch 1 Mistakes Avoided
- [x] Not preachy — narrator is confused and learning, not lecturing; the heist carries the lesson
- [x] Real narrative arc — lost in tasks → Ocean's Eleven reveals what alignment looks like → "Now I see the line"
- [x] Emotional — "who-knows-where," "standing still," "from my hands to why I'm here"
- [x] Works as a song without knowing Lenny — the heist metaphor and "trace it back" hook are universal
- [x] No filler rhymes — every line earns its place
- [x] No insider jargon — Ocean's Eleven and TED are universally known references
