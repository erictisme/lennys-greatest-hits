# Lenny's Greatest Hits

A music web app turning Lenny Rachitsky's best podcast and newsletter insights into catchy, playable songs.

## Audio Files

All audio files must be in `public/audio/` and committed to git. **Never add `public/audio` to `.gitignore`.**

The MP3 files are served directly by Vercel from the `public/` directory. If they're gitignored, they won't be deployed and all songs will be broken in production.
